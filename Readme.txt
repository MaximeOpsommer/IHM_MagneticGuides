TP2 - IHM - Delassus Alexandre - Opsommer Maxime

Nous avons été autorisé à réaliser les TPs en binome, nous aurons donc le même rendu.

Pour ce TP, nous n'avons pas respecter les Tags car ceux-ci ne fonctionner pas.
Nous avons quand même fini ce TP en utilisant des listes de MagneticGuides et
des maps de CShape et de MagneticGuides pour palier au Tag.

Dans le mouvement des guides, si vous allez trop vite, il y a une erreur, je n'ai
pas réussi à regler ce problème.

Utilisation :
	Click Gauche sur le vide	-> Créer un guide horizontal
	Click Droit sur le vide 	-> Créer un guide vertical
	Click Milieu sur le vide 	-> Cacher les guides
	Release le click milieu		-> Voir les guides
	Click Droit sur un objet	-> Le déplace
